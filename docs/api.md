### 结构树

```shell
.
├── __init__.py
├── _except.py              # 异常
├── config.py               # 框架配置
├── ctx.py                  # 框架上下文
├── frame.py                # frame框架
├── generics.py             # 通用类视图
├── local.py                # 框架本地线程对象
├── logger.py               # 框架通用日志文件库
├── route.py                # 框架路由
├── scaffold.py             # frame框架骨架
├── serizlizer.py           # 通用类视图序列化工具库
├── server.py               # 框架提供的内置server
├── sessions.py             # 框架提供对session的支持
├── testing.py              # 框架提供的对单元测试工具
├── utils.py                # 框架提供的部分工具
├── view.py                 # 类视图
├── wrappers.py             # 框架对请求、响应的包装器
└── wsgi.py                 # 框架对wsgi的支持
```
